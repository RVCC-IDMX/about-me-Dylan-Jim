# About Me Project

My name is Dylan Jimenez, and this is my final About Me project for Web Dev I. This website is meant to show who I am, some of my interests, and the web design skills I learned. The site includes a bio section, hobbies section, photo gallery, blog article, embedded video, and contact form.

The purpose of this project is to practice building a complete webpage using HTML and CSS. I used HTML to organize the page into different sections and CSS to style the colors, layout, images, forms, and media. I also worked on making the page look better on different screen sizes by using media queries.

## Color Scheme

My website uses a light yellow background with blue text and light blue sections. The main colors are:

- Background: `#FAF9C8`
- Text: `#286AA1`
- Section background: `#DDFDF4`
- Form/button blue: `#008cff`

I chose these colors because they make the website feel bright and simple. The blue also gives the page a clean look, and the yellow background makes it stand out from a plain white page.
